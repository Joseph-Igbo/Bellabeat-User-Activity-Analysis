# Bellabeat-Capstone-Analysis
End-to-end analysis of Fitbit user activity data to generate engagement and product growth insights for Bellabeat using SQL and R.

## Project Overview
This project examines user activity patterns to inform product strategy and engagement initiatives. Analysis include:

- Differences in activity between weekdays and weekends  
- Relationship between activity intensity, steps, and calories burned  
- Segmentation of users by activity levels  

The insights translate into actionable business recommendations designed to improve user retention and guide feature development.

## How to Explore the Project

- **Live Interactive Report:** Accessible via GitHub Pages: [View Report](https://joseph-igbo.github.io/Bellabeat-Capstone-Analysis/)  
  Contains all the plots, tables, insights, and strategic recommendations in an executive-ready format.  

- **R Markdown File:** `R/Bellabeat_Capstone.Rmd` – can be knitted locally to reproduce the report.  

- **SQL Cleaning Script:** `sql/data_cleaning.sql` – demonstrating the end-to-end data preparation workflow from raw CSVs to analysis-ready dataset.
